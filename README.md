## Tests

1. Ensure Firefox Nightly is installed on your machine.
2. `$ npm install`
3. `$ npm run test-unit`

If your would like tests to run on file change use:

`$ npm run test-unit-dev`

## Lint check

Run lint check with command:

`$ npm run test-lint`
